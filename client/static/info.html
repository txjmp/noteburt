Noteburt

  • A tool for creating, organizing, and sharing notes.
  • Notes are stored in a notebook.
  • Conceptually like a real "paper" notebook.
  • Anyone can view a notebook's contents.

Super Simple

To View a Notebook

  1. Select "View"
  2. Enter notebook name (caps, spaces, commas, etc. ignored)
  3. Click "Open"

To Create a Notebook

  1. Select "Create"
  2. Enter notebook name
  3. Get "Access Code" (needed to edit notebook later)

Adding Notes

  1. Select Tab where note is to be added & click "Add Note"
  2. Specify content type (text, html, markdown)
  3. Check "View" to see how note will be displayed to others

Content Options

Select From

  1. Plain text (the default), just start typing
  2. Markdown - an easy to use method of spicing up your content
  3. HTML - content limited to your imagination
